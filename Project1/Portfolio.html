Portfolio
Multiplier Device
  • In this project, I synthesized a multiplier hardware onto an FPGA using verilog programming on the Vivado platform. To interact with this multiplier device, I compiled linux for FPGA and wrote device drivers that would allow the operating system to communicate with the multiplier. Through this project, I learned about device driver development and how to practically apply devices I make or program.
FPGA Projects
  • I have worked frequently with the Zybo Z7-10 board and have made numerous projects for it using Verilog. One such project was a traffic light controller system, in which hardware was synthesized constituting a finite state machine that would change the state of a light, here represented by the leds on the board, based on input from a clock and a sensor. Projects like this one have allowed me to practice skills in hardware design and programming as well as state machine logic, and have helped build my understanding of digital logic at the hardware level.
Memory Allocator
  • I programmed a Buddy System Memory Allocator in C++. Though managing the memory was difficult, it allowed me to practice memory operations and better understand how memory can be allocated and managed by an operating system or program. The knowledge I gained from this helps me in managing memory in devices, such as the aforementioned FPGA multiplier, as well as in other aspects of my programming.
